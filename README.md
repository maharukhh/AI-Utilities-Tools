# AI Utilities & Tools

A collection of Python-based utility programs and tools designed to solve everyday programming tasks, automation problems, and basic AI/logic-based applications.

This repository is continuously expanding with new tools and features.

## About This Repository

This repository contains multiple utility-based projects built using Python. Each project is independent and focuses on solving a specific problem such as calculations, security checks, automation, and intelligent systems.

These tools demonstrate concepts from:

* Python Programming
* Problem Solving
* Basic Artificial Intelligence
* Data Processing
* Automation
* Logic Building

New tools will be added regularly.

## Projects Included (Current)

### 1. Simple Calculator

A command-line calculator that performs basic and advanced arithmetic operations.

* Operations: +, -, *, /, %, ^
* Interface: CLI
* Concepts: Functions, expressions, input handling

### 2. Password Strength Checker

A tool that evaluates password strength based on security rules.

* Checks: Length, patterns, complexity
* Logic: Rule-based evaluation
* Output: Strength score + suggestions

### 3. AI Chatbot

A simple chatbot that responds to user input using basic NLP concepts.

* Features: Intent matching, response generation
* Concepts: Natural Language Processing basics

### 4. Robot Control System

A simulation-based system for controlling robot movements.

* Features: Movement commands, basic control logic
* Concepts: Automation, simulation logic
