# Simple Calculator

A basic Python command-line calculator that performs simple arithmetic operations between two numbers.

## About This Project

This project is a beginner-level calculator built using Python. It takes two numbers and an operator as input and returns the result.

It demonstrates basic programming concepts such as:

* User input handling
* Conditional statements
* Arithmetic operations
* Error handling

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Handles division by zero
* Simple command-line interface

## Technologies Used

* Python

## How It Works

* User enters first number
* User enters operator
* User enters second number
* Program performs calculation based on operator
